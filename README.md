# Smart Online Shopping System


## Project Description

Smart Online Shopping is an e-commerce website that allows customers to browse products, add items to cart, place orders, and manage purchases online.



## Features

- User registration and login
- Product browsing
- Product categories
- Product search
- Shopping cart
- Checkout system
- Order management
- Customer order history
- Admin dashboard
- Product management
- Order status management



## Technologies Used

Frontend:
- HTML
- CSS
- Bootstrap 5
- JavaScript


Backend:
- PHP


Database:
- MySQL


Server:
- XAMPP Apache Server
